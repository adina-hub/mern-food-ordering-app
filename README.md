# Food ordering platform with REACT.JS! (Node.js, MongoDB, Auth, Stripe & More)

# Features
* Secure Authentication: Implemented with Auth0 for reliable and secure sign-in/sign-out functionality.

* Add and Manage Restaurants: Allows users to add new restaurants, edit existing ones, and manage restaurant details.

* Order Placement and Management: Users can place orders, pay using Stripe, and update order statuses.

* Payment Processing: Integrated Stripe for secure and efficient payment transactions.

* Image Upload and Management: Utilized Cloudinary for managing restaurant images.

* Deployment: Hosted on Render for both frontend and backend services ensuring scalability and performance.

Live app: https://mern-food-ordering-app-frontend-sgff.onrender.com
